Statement Report FX
Statement Report FX is a JavaFX desktop application that allows users to generate the four primary financial statements individually or complete a full set of financial reports one statement at a time.

Designed for accounting students and professionals, the application performs automatic calculations, validates financial data, and helps ensure financial statements are accurate before completion.

Features
-Generate financial statements individually
- Automatic financial calculations
- Built-in input validation
- Balance Sheet verification
- Clean and easy-to-use JavaFX interface

Financial Statement Generators
1. Income Statement
Create an Income Statement by entering:
- Company Name
- Revenue accounts
- Expense accounts

The application automatically calculates:
- Net Income (Profit or Loss)

2. Statement of Retained Earnings
Generate a Statement of Retained Earnings by entering:
Beginning Retained Earnings
Net Income
Dividends

The application automatically calculates:
 Ending Retained Earnings

3. Balance Sheet
Enter:
- Assets
- Liabilities
- Owner's Equity

The application automatically verifies the accounting equation:
Assets = Liabilities + Equity
If the equation balances, the statement is complete.

4. Statement of Cash Flows
Record cash transactions under:Operating Activities
- Investing Activities
- Financing Activities

The application automatically calculates:

Net Change in Cash
Getting Started
Step 1 —Open the Dashboard
Launch the application and select the financial statement you want to generate.
Step 2 — Create an Income Statement
1. Enter the company name.
2. Input all revenue accounts.
3. Input all expense accounts.
4. Review the automatically calculated **Net Income**.

Step 3 — Create Additional Statements
Statement of Retained Earnings
Enter:
- Beginning Retained Earnings
- Net Income
- Dividends

The program calculates Ending Retained Earnings automatically.

Statement of Cash Flows

Enter cash transactions for:

- Operating Activities
- Investing Activities
- Financing Activities

The program calculates the total change in cash.

Step 4 — Complete the Balance Sheet

Enter:
- Assets
- Liabilities
- Equity
Click verify to confirm:
Assets = Liabilities + Equity

If the statement balances, your Balance Sheet is complete.

Built-in Error Checking
Statement Report FX includes several validation features to reduce errors while preparing financial statements.
Invalid Input Detection

- Prevents letters or invalid characters from being entered into numeric fields.
- Displays helpful error messages when incorrect values are entered.

Balance Sheet Verification

The application checks whether:
Assets = Liabilities + Equity    If the equation does not balance, a warning is displayed so the values can be corrected before completing the report.
